


                  



                                               PROJETO DE CONSULTA À TABELA FIPE




PRIMEIRAMENTE O PROGRAMA DEVE EXIBIR QUAL TIPO DE VEÍCULO VAMOS CONSULTAR (CARRO, MOTO, CAMINHÃO)
---- requisição: https://parallelum.com.br/fipe/api/v1/carros/marcas //para motos /motos e caminhões /caminhoes

QUANDO O USUÁRIO ESCOLHER O TIPO DO VEÍCULO, O PROGRAMA FARÁ UMA REQUISIÇÃO À API E LISTARÁ TODAS AS MARCAS DISPONÍVEIS
PARA CONSULTA
E O USUÁRIO DEVERÁ ESCOLHER UMA E DIGITAR O CÓDIGO DA MESMA
EX: -------------------  https://parallelum.com.br/fipe/api/v1/carros/marcas/21/modelos (FIAT)
MODELAR E ORDENARA OS CÓDIGOS EM ORDEM NUMÉRICA:

LOGO DEPOIS O PROGRAMA EXIBIRÁ TODOS OS MODELOS DE CARROS DAQUELA MARCA, O USUÁRIO DEVE DIGITAR UM TRECHO DO NOME DO
MODELO DESEJADO, E O PROGRAMA DEVE LISTAR TODOS OSMODELOS QUE CONTENHAM AQUELE NOME, PARA ASSIM PASSAR O CÓDIGO DO
MODELO DESEJADO AO PROGRAMA , QUE POR SUA VEZ DEVE ENCAMINHA-LO À API
EX: ----------------------------------- https://parallelum.com.br/fipe/api/v1/carros/marcas/21/modelos/5448/anos

POSTERIORMENTE DEVERÁ SER EXIBIDO TODOS OS ANOS DISPONÍVEIS PARA A CONSULTA DAQUELE RESPECTIVO MODELO. O PROGRAMA DEVE
ITERAR SOBRE A LISTA DEVOLVIDA PELA API E FAZER UMA REQUISIÇÃOCOM CADA ANO DISPONÍVEL, ARMAZENANDO O RESULTADO DEVOLVIDO
 E EXIBINDO AO USUÁRIO.
